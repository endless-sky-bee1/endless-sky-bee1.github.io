# 产品概述

> 本章介绍 Endless 智能摄像头的定位、核心能力与典型应用场景。

## 产品定位

Endless 是一套基于海思 3516cv610 平台的开源智能摄像头软件方案，提供从内核驱动到 Web 前端的完整全栈实现，支持软硬件定制。

## 核心能力

| 能力 | 说明 |
|------|------|
| 流媒体 | RTSP / RTMP / WebRTC / HTTP-FLV，WebRTC 延迟 < 500ms |
| AI 分析 | 人脸、人形、车辆、宠物检测，支持自定义 YOLO 模型 |
| 云端 AI | 接入阿里云通义千问 VL，支持图像问答与语音对话 |
| 存储 | SD 卡录像、循环覆盖、事件触发录像、回放检索 |
| 协议 | ONVIF、GB28181 标准接入 |
| 安全 | HTTPS、JWT 认证、用户权限管理 |

## 系统架构

```
应用层        Web界面 · RESTful API · ONVIF · GB28181
基础业务层    音视频存储 · 设备升级 · AI算法管理
基础软件层    GStreamer · FFmpeg · Live555 · SQLite
硬件抽象层    音视频编解码 · AI硬件加速 · USB接口
内核层        Linux内核 · 设备驱动
硬件层        海思3516cv610 · aarch64
```

## 典型应用场景

- 智能安防监控
- 工业视觉检测
- 楼宇门禁对讲
- 边缘 AI 推理设备
